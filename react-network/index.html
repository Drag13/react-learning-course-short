Internet Explorer

Despite it's being deprecated even by the author, it's still popular in a big companies

Fetch doesn't work in IE

Canceling request

You are not able to cancel request issued by fetch. (experimental support is on the way)

No progress tracking, no timeout

This features also missing right now

Fetch is poorly supported in the ServerSide

To work with fetch from Node.JS you need at least 17.5 version
