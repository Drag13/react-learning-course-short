What is a Pet Project

What is the main difference between Pet and Learning Project?

Main goal for a learning project - is to genereate some knowledge

Main goal for a pet project - is to be released

Considering Pet project - always think about the release

Hot to pick a theme

Good theme is super important

The good theme will suport you during the development and attract other people after the release

I know two options to pick a good theme

Joy themes

Pain themes

Planing

When doing pet project you will not have much time for a detailed planning

  • Set one (maximum two) goals
  • Note nice to have features somewhere just not to forget
  • Categorize all work as must have (the solution is usesless without such features) and nice to have
  • Implement only must have features

Pitfalls

Underestimates

The first pet project should be something around two weeks 2 hours each day of work

Adding non crucial features

Implement only that features that absolutely needed for the project

Pausing

Better to work 2 hours each day rather than 6 -> 4 -> 2 -> 1 -> 0 -> 1 -> 0 -> 0
